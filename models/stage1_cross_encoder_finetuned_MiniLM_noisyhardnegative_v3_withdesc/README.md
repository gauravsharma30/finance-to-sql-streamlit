# CrossEncoder based on cross-encoder/ms-marco-MiniLM-L6-v2

This is a [Cross Encoder](https://www.sbert.net/docs/cross_encoder/usage/usage.html) model finetuned from [cross-encoder/ms-marco-MiniLM-L6-v2](https://huggingface.co/cross-encoder/ms-marco-MiniLM-L6-v2) using the [sentence-transformers](https://www.SBERT.net) library. It computes scores for pairs of texts, which can be used for text reranking and semantic search.

## Usage

### Direct Usage (Sentence Transformers)

First install the Sentence Transformers library:

```bash
pip install -U sentence-transformers
```

Then you can load this model and run inference.
```python
from sentence_transformers import CrossEncoder

# Download from the ðŸ¤— Hub
model = CrossEncoder("cross_encoder_model_id")
# Get scores for pairs of texts
pairs = [
    ['What are we spending to keep the business running this year?', 'Trade Payables can be defined as Amounts due to suppliers of goods or services.'],
    ['Are we giving investors a good return via dividends?', 'Trade Payables can be defined as Amounts due to suppliers of goods or services.'],
    ['How much are things like our patents worth?', 'Borrowings can be defined as Loans from banks and financial institutions, classified as current or non'],
    ['How quick are we getting paid by customers?', 'Property, Plant and Equipment (PPE) can be defined as Tangible items held for use in the production or supply of goods or services, or for administrative purposes (Ind AS 16).'],
    ['How has our provit after production been trending lately?', 'Gross Profit can be defined as Revenue - COGS.'],
]
scores = model.predict(pairs)
print(scores.shape)
# (5,)

# Or rank different texts based on similarity to a single text
ranks = model.rank(
    'What are we spending to keep the business running this year?',
    [
        'Trade Payables can be defined as Amounts due to suppliers of goods or services.',
        'Trade Payables can be defined as Amounts due to suppliers of goods or services.',
        'Borrowings can be defined as Loans from banks and financial institutions, classified as current or non',
        'Property, Plant and Equipment (PPE) can be defined as Tangible items held for use in the production or supply of goods or services, or for administrative purposes (Ind AS 16).',
        'Gross Profit can be defined as Revenue - COGS.',
    ]
)
# [{'corpus_id': ..., 'score': ...}, {'corpus_id': ..., 'score': ...}, ...]
```
